#### 简介

HMWP是从中国k12数学问题库中提取的共5491个中文数学题，包括2955道一元线性问题、1636道二元线性问题和900道一元非线性问题。测试样本有500个。

```json
	"equation": "5.0 * x + 3.0 * y - 2.0 = 50.0 ; ( 11.0 * x + 5.0 * y ) * 0.9 = 90.0",
	"ans": [
		5,
		9
	]
}
```
