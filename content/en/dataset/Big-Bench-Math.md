#### Introduction

Big-Bench (Math) is a set of four basic arithmetic problems, involving addition, subtraction, multiplication, and division, with numbers ranging from 1 to 5 digits.

#### Example

```json
{
  "input": "What is 0 plus 1?",
  "target": "1",
  "target_scores": {
    "5": 0.0,
    "4": 0.0,
    "328": 0.0,
    "banana": 0.0,
    "house": 0.0,
    "1": 1.0
}
```
